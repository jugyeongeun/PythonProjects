AI 단어학습 게임
#기획구조(기능, 설명)
1.단어퀴즈: 뜻 --> 영어 단어 입력(또는 반대)
2.난이도 설정: Easy / Normal / Hard 단어풀 분리
3.오답 저장: 틀린단어 저장 --> 복습 모드 지원
4. 추천 기능: 자주 틀리는 유형 기반으로 단어 추천

#파일구조
Word_Game
  |-- main.py       #실행파일
  |-- word_data.py  #단어/난이도 데이터
  |-- logic.py      #문제 출제, 채점, 오답 저장
  |-- ui.py         #tkinter UI 구성 (toolkit interface)
